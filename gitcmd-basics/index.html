Git and GitHub Overview

Git is a software tool and a version control system.

GitHub is a web-based Git repository hosting service. It is a popular platform for hosting and managing version-controlled projects.

Alternatives to GitHub

  • GitLab
  • BitBucket
  • Azure Repos
  • Gitea

Common Git Commands

Any project files can be added to the Git repository using the terminal with the commands below:

git add .

Adds all files in the current directory to the staging area.

git commit -m "message"

Commits the changes in the staging area with a message describing the changes.

git branch

Lists all the branches you have created.

PWD

Displays the present working directory.

git checkout "branch"

Switches to a different branch in the repository.

git merge

Merges one branch into another in the repository.

git status

Checks the current status of the working directory and staging area.

git add <filename>

Adds a specific file to the staging area.

git branch -d <branch>

Deletes the specified branch.

git diff

Shows changes between the working directory and the staging area.

git diff --staged

Shows changes between the last commit and the staging area (staged changes).

git stash

Temporarily stashes uncommitted changes (both staged and unstaged).

git stash list

Shows a list of all stashed changes.

git stash apply stash@{0}

Applies a specific stash by its index.

git stash pop

Applies the last stash and removes it from the stash list.

git stash clear

Clears all stash entries.

git rebase

Moves or combines a sequence of commits to a new base commit.

git branch <light-mode>

Creates a new branch named light-mode.

git reflog

Displays a list of all recent branch changes and commits.
